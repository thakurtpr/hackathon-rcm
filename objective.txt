## STEP 10 — FACE MATCH PIPELINE

Write full implementation of `ai_service/app/pipelines/face_match_pipeline.py`.

```python
async def run(
    aadhaar_path: str,
    selfie_path: str,
    minio_client,
    insightface_app,   # globally loaded FaceAnalysis instance
    qdrant_service,
    user_id: str,
    app_id: str
) -> FaceMatchResult:
```

Full logic:
1. Fetch both files from MinIO concurrently with asyncio.gather
2. Convert each bytes → PIL Image → np.array (BGR for InsightFace: cv2.cvtColor(img, cv2.COLOR_RGB2BGR))
3. `faces_a = await asyncio.to_thread(insightface_app.get, aadhaar_np)`
4. `faces_b = await asyncio.to_thread(insightface_app.get, selfie_np)`
5. If either list is empty: return FaceMatchResult(face_match_score=0.0, face_match_pass=False, flag="no_face_detected")
6. emb_a = faces_a[0].embedding (512-dim numpy array) — normalize: emb / np.linalg.norm(emb)
7. emb_b = faces_b[0].embedding — normalize same
8. score = float(np.dot(emb_a, emb_b))  # cosine sim of unit vectors
9. Thresholds:
   - score >= 0.85 → flag="passed", face_match_pass=True
   - 0.70 <= score < 0.85 → flag="manual_review", face_match_pass=False
   - score < 0.70 → flag="failed", face_match_pass=False
10. Store selfie embedding in Qdrant "face_embeddings" (dim=512, Cosine):
    id = hashlib.sha256(user_id.encode()).hexdigest()
    payload = {"user_id": user_id, "app_id": app_id}
    await qdrant_service.upsert("face_embeddings", id, emb_b.tolist(), payload)
11. Return FaceMatchResult(face_match_score=round(score, 4), face_match_pass=face_match_pass, flag=flag)

Wrap entire function in try/except — on any error return FaceMatchResult(score=0.0, pass=False, flag="failed")

```python
    aadhaar_number: str,   # raw Aadhaar — hash before sending
    mobile: str,
    doc_ids: list[str],
    selfie_embedding: list[float],   # 512-dim from face match step
    backend_client,
    qdrant_service
) -> FraudResult:
```

Implement each check as a separate async function, run all 5 via asyncio.gather:

1. `check_duplicate_pan(pan_number, backend_client) → FraudCheck`:
   - hash = hashlib.sha256(pan_number.upper().encode()).hexdigest()
   - exists = await backend_client.check_pan(hash)
   - FraudCheck(check_name="duplicate_pan", passed=not exists, reason="PAN already registered" if exists else None)

2. `check_duplicate_aadhaar(aadhaar_number, backend_client) → FraudCheck`:
   - hash = hashlib.sha256(re.sub(r'\s', '', aadhaar_number).encode()).hexdigest()
   - FraudCheck(check_name="duplicate_aadhaar", passed=not exists, ...)

3. `check_face_pool(user_id, selfie_embedding, qdrant_service) → FraudCheck`:
   - results = await qdrant_service.search("face_embeddings", selfie_embedding, limit=5, score_threshold=0.92)
   - fraud = any(r.payload["user_id"] != user_id for r in results)
   - FraudCheck(check_name="face_pool_duplicate", passed=not fraud, reason="Face matches another account" if fraud else None)

4. `check_doc_metadata(doc_ids, backend_client) → FraudCheck`:
   - For hackathon: call backend_client GET /documents/{doc_id}/status for each doc
   - Check if doc status shows any tamper flag in returned metadata
   - FraudCheck(check_name="doc_metadata", passed=True if no tamper else False, ...)

5. `check_velocity(user_id, backend_client) → FraudCheck`:
   - count = await backend_client.get_app_count(user_id, days=30)
   - fraud = count > 2
   - FraudCheck(check_name="velocity_check", passed=not fraud, reason=f"{count} applications in 30 days" if fraud else None)

Aggregation:
- Hard fails (fraud_flag=True if ANY of checks 1, 2, or 3 failed)
- fraud_reasons = [c.reason for c in checks if not c.passed and c.reason]
- fraud_confidence = round((len([c for c in checks if not c.passed]) / 5) * 100, 2)
- Return FraudResult(app_id=app_id, fraud_flag=fraud_flag, fraud_reasons=fraud_reasons, fraud_confidence=fraud_confidence, checks=checks)
